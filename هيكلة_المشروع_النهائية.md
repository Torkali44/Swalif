# سين جيم | الإمارات — الهيكلة النهائية للمشروع (Laravel + Blade)

> هذا الملف بديل نهائي واحد يجمع أفضل ما في النماذج الأربعة اللي راجعتها، ويحسم كل التعارضات بينها (تسمية الجداول، تنظيم الـ Services، تفاصيل لوحة التحكم). اعتمدت بشكل أساسي على تفاصيل **النموذج 4** لأنه الأكثر اكتمالًا (Console Commands، فصل Controllers حسب النوع، Services مقسّمة بالدومين، أمثلة كود فعلية)، وسددت منه الثغرات بحاجات من باقي النماذج.

---

## 1. مبادئ التصميم (قبل ما تقرأ الشجرة)

1. **الفئات بيانات (DB) مش فولدرات كود.** "كل فئة = فولدر" في النماذج التانية معناها الصح: *لكل فئة ملف Seeder منفصل + أيقونة منفصلة في `resources/images/categories/`* — مش Controller أو Model منفصل. الفئة نفسها record واحد في جدول `categories`، عشان تقدر تضيفها من لوحة التحكم بدون نشر كود جديد.
2. **الصعوبة والنقاط مرتبطين لكن منفصلين.** كل سؤال له `level` (easy/medium/hard) و`points` (200/400/600). العلاقة التقليدية: صف 200 = سهل، 400 = متوسط، 600 = صعب — لكن خزّنهم كحقلين منفصلين في لوحة التحكم عشان تقدر تغيّر القيمة المالية للنقطة بدون ما تغيّر مستوى الصعوبة.
3. **وسائل المساعدة تُخزَّن JSON على الفريق مش جدول منفصل.** (فض / اتصال بصديق / جاوب جواين) — تبسيط مقصود لأنك قلت "عاوز أعمل الأساسيات بس مش كله". لو احتجت لاحقًا تتبّع تفصيلي لكل استخدام، سهل تفصلها لجدول `game_team_helpers` بدون ما تكسر الموديل.
4. **Services مقسّمة بالدومين مش كلها في مجلد واحد** (Game / Category / Subscription / Payment) — كل دومين بيتوسع لوحده بمرور الوقت.
5. **بوابة الدفع Interface من أول يوم** (`PaymentGatewayInterface`) — حتى لو هتشتغل ببوابة واحدة بس دلوقتي، يخليك تبدّلها بدون لمس الـ Controllers.

## 3. الجداول (Database Schema)

| الجدول | الأعمدة الأساسية |
|---|---|
| **categories** | `id`, `name_ar`, `name_en`, `slug`, `group` (uae/general), `icon`, `description`, `is_active`, `sort_order`, timestamps |
| **questions** | `id`, `category_id`, `level` (easy/medium/hard), `points` (200/400/600), `question_text`, `image`, `audio`, `time_limit` (ثواني), `is_active`, timestamps |
| **question_options** | `id`, `question_id`, `option_text`, `is_correct` |
| **games** | `id`, `user_id`, `category_id`, `name`, `status` (pending/playing/finished), `winner_team_id`, `started_at`, `ended_at` |
| **teams** | `id`, `game_id`, `name`, `score`, `helpers_left` (json: `swap`, `phone_friend`, `two_answers`) |
| **game_questions** | `id`, `game_id`, `question_id`, `assigned_team_id` (nullable), `points_awarded`, `answered_correctly`, `answered_at` |
| **plans** | `id`, `name`, `type` (weekly/monthly/yearly), `price`, `currency` (AED), `duration_days`, `features` (json), `is_active`, `is_recommended`, `sort_order` |
| **subscriptions** | `id`, `user_id`, `plan_id`, `starts_at`, `ends_at`, `status` (active/expired/cancelled), `payment_id` |
| **payments** | `id`, `user_id`, `subscription_id`, `gateway` (stripe/tap), `gateway_reference`, `amount`, `currency`, `status`, `meta` (json) |

**مثال Migration لجدول الأسئلة:**
```php
Schema::create('questions', function (Blueprint $t) {
    $t->id();
    $t->foreignId('category_id')->constrained()->cascadeOnDelete();
    $t->text('question_text');
    $t->string('image')->nullable();
    $t->string('audio')->nullable();
    $t->string('level');                          // easy | medium | hard
    $t->unsignedSmallInteger('points');            // 200 | 400 | 600
    $t->unsignedSmallInteger('time_limit')->default(60);
    $t->boolean('is_active')->default(true);
    $t->timestamps();
});
```

**مثال Controller لحفظ سؤال (Admin):**
```php
public function store(StoreQuestionRequest $request)
{
    $data = $request->validated();
    // ['category_id','question_text','level','points','time_limit','image?','audio?','options' => [...]]

    $question = Question::create($data);
    foreach ($data['options'] as $opt) {
        $question->options()->create($opt); // ['option_text' => ..., 'is_correct' => ...]
    }

    return back()->with('ok', 'تم حفظ السؤال بنجاح');
}
```

## 6. هوية الألوان (CSS Variables في `tokens.css`)

الألوان دي هي نفسها المستخدمة فعليًا في ملفات الـ Home Page (`index.html` / `style.css`) اللي جهزتها قبل كده — خليها المرجع الوحيد عشان كل الشاشات تطلع متناسقة:

```css
:root{
  --night-950:#0B1E22;   /* خلفية أساسية: ليل الصحراء */
  --night-900:#102C31;
  --night-800:#163940;
  --teal-700:#1F5C5E;    /* أخضر نخيل عميق */
  --gold-500:#C9A24B;    /* ذهب الكثبان — اللون المميز */
  --gold-300:#E4C57C;
  --sand-200:#EAD9AE;
  --maroon-600:#7C2E2A;  /* عنابي تراثي — استخدام نادر */
  --cream-50:#F8F3E7;
  --ink-900:#1C1712;
}
```

> ملاحظة: كان فيه اقتراح لألوان "نيون" حماسية (أحمر/أخضر فسفوري) في أحد النماذج القديمة — تم استبعادها لأنها بتبعد عن الطابع الفخم/التراثي المطلوب، وبتبقى مكررة مع تصاميم AI عامة. الباقة الحالية (ليل + ذهب + أخضر نخيل) أنسب لهوية "سين جيم الإمارات".
